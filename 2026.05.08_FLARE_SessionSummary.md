# FLARE Motor Debug Session — 2026-05-08

## Problem
Motors pulse/stutter randomly; spin direction inconsistent.

## Root Causes Found and Fixed

### 1. Throttle mapping floor too high (`main.c`)
RC minimum (1000) was mapping to DSHOT 1200 instead of DSHOT 48, causing motors to immediately blast at ~58% power when armed.

**Fixed to:**
```c
uint16_t dshot_thr = (uint16_t)(((float)(rc_pkt.throttle - 1000) / 1000.0f) * (2047.0f - 48.0f) + 48.0f);
```

### 2. PID engage threshold wrong (`flare.c`)
Threshold of `1300` was 62% of the corrected DSHOT range, meaning PID stabilisation never engaged during normal flight.

**Fixed to:** `if (state.throttle < 250)` (~10% above DSHOT minimum)

### 3. Arm switch inverted (`esp32_remote/src/main.cpp`)
`INPUT_PULLUP` with `== HIGH` meant the FC was always commanded ARMED when the switch was off.

**Fixed to:**
```cpp
pkt.armed = (digitalRead(PIN_ARM_SWITCH) == LOW) ? FLARE_ARMED : FLARE_DISARMED;
```

### 4. Excessive blocking UART in main loop (`main.c`)
~35ms of `HAL_UART_Transmit` per iteration was dropping the effective control rate from 100Hz to ~22Hz. Removed all `[DEBUG]` prints and the `[RC] bytes=...` diagnostic line. One `[THR]` print remains per armed iteration.

## Still Outstanding

### Motor direction
All 4 ESCs have `Eep_Pgm_Direction=1` (same direction). Need to verify with props off — M1/M3 should spin CCW, M2/M4 CW. Fix via:
- Physical wire swap on CW motors (swap any two of three phase wires on M2 and M4), OR
- Set `Eep_Pgm_Direction=2` for ESC2/ESC4 in BLHeliSuite and reflash those ESCs

### Throttle calibration verification
Throttle stick center sits at ~43% on the OLED. Need to push stick to physical bottom stop and confirm display reads ~0%. If it reads significantly above 0%, `kThrottle.min = 265` in `esp32_remote/src/main.cpp` needs updating to the actual ADC floor on pin A1.

### Magnetometer calibration
Yaw drifts ~65 dps when motors run due to EMI corrupting the QMC5883P. Hard-iron offsets (`mag_cal.offset_x/y` in `main.c`) are currently zero. Needs calibration before flight; consider physically separating the magnetometer from power leads and ESC wiring.
